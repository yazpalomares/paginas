Historia de los Lenguajes de Programacion

JAVA

Java es un lenguaje de programación de alto nivel desarrollado por Sun Microsystems (ahora propiedad de Oracle) en la década de 1990. Fue creado por James Gosling y su equipo con el objetivo de ser un lenguaje portable y orientado a objetos. Java se hizo popular rápidamente debido a su capacidad para ejecutarse en diferentes plataformas, lo que se logró a través de la máquina virtual de Java (JVM). Uno de los hitos más importantes en la historia de Java fue su introducción en 1995. Desde entonces, Java se ha utilizado ampliamente en el desarrollo de aplicaciones empresariales, aplicaciones móviles (Android), aplicaciones web y más. La plataforma Java se ha expandido constantemente, incluyendo la introducción de bibliotecas y frameworks como Spring y Java EE. Java ha sido un pilar en el mundo de la programación durante décadas.

Ejemplo Hello Word Java

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, World!"); } }

PYTHON

Python es un lenguaje de programación de alto nivel creado por Guido van Rossum a finales de la década de 1980. Van Rossum diseñó Python con un enfoque en la legibilidad del código y la productividad del programador. El nombre del lenguaje, Python, se inspiró en el grupo de comedia británico Monty Python, del que van Rossum era fanático. Python se ha ganado una gran popularidad debido a su sintaxis clara y sencilla, lo que lo hace ideal para principiantes. A lo largo de los años, Python ha evolucionado y se ha convertido en uno de los lenguajes más utilizados en una variedad de aplicaciones, incluyendo desarrollo web (Django y Flask), ciencia de datos (con bibliotecas como NumPy y Pandas) y automatización de tareas.

Ejemplo Hello Word Python

print("Hello Word")

PHP

PHP (Hypertext Preprocessor) es un lenguaje de programación de servidor ampliamente utilizado para el desarrollo web. Fue creado por Rasmus Lerdorf a mediados de la década de 1990. Originalmente, PHP se diseñó como un conjunto de scripts para manejar la información de su sitio web personal, pero con el tiempo se convirtió en un lenguaje de programación completo. PHP se utiliza para generar contenido web dinámico, manejar formularios, interactuar con bases de datos y mucho más. A lo largo de los años, se han desarrollado numerosos frameworks y CMS (Content Management Systems) basados en PHP, como WordPress y Drupal, que han contribuido a su popularidad en el desarrollo web.

REACT

React es una biblioteca de JavaScript desarrollada por Facebook, lanzada en 2013. Aunque no es un lenguaje de programación en sí, React es esencial para el desarrollo de aplicaciones web modernas. React se utiliza para crear interfaces de usuario interactivas y dinámicas, y su popularidad ha crecido rápidamente. React se basa en la idea de componentes reutilizables que pueden actualizarse de manera eficiente en respuesta a cambios de datos. Esta biblioteca se ha convertido en una parte integral del desarrollo web moderno, y se utiliza en conjunto con otros lenguajes y tecnologías como JavaScript, HTML y CSS.

Ejemplo Hello Word React

import React from 'react'; class HelloWorld extends React.Component { render() { return ( <div> <h1>Hello, World!</h1> </div> ); } } export default HelloWorld;
